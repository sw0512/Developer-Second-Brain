# Hooks (v0.4 — 예정)

이 폴더는 **자동 탐지 훅**을 위한 자리입니다. v0.1에서는 비활성입니다.

## 계획

Claude Code의 훅(예: `Stop` — 대화가 끝날 때)을 이용해, 사용자가 아무것도 하지 않아도
방금 한 작업이 기록할 가치가 있는지 자동으로 판단하고 **제안**까지 띄웁니다.

```
개발 종료(Stop)
   │
   ▼
탐지 규칙으로 기록 가치 판단 (detection-rules.md)
   │
   ▼
가치 있으면 → "기록할까요?" 제안
```

## 유지되는 원칙

- 훅은 **제안까지만** 자동화합니다. **저장은 여전히 사용자 승인 필수** — 자동 저장 금지.
- 판단 로직은 스킬의 `references/`를 재사용합니다 (중복 없음).

## 예정 구성

- `hooks.json` — 훅 이벤트 → 실행 매핑.
- 판단 스크립트 또는 프롬프트 훅.

v0.4에서 스킬 본체 변경 없이 이 폴더에 훅만 추가하는 것이 목표입니다.
